Final accepted configuration

- Keep:
  - bounded BFS cache reuse
  - predecessor-based shortest-path unions
  - direct CSR construction
  - graph-local structural cache
- Revert:
  - cheap structural hash replacement
- Result:
  - learning-curve quality matches the original benchmark when using the original `sha256` hash
  - end-to-end runtime improves from about `74.91–80.77s` to about `40.40–42.21s` at train size `5599`
